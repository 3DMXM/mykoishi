三阶魔方

## 命令
|  方法  | 命令  |
|  ----  | ----  |
| 开始游戏  | cb\|cube\|魔方 |

### 描述
三阶魔方
[nonebot-plugin-cube](https://github.com/initialencounter/nonebot-plugin-cube)重写版
### 使用用法
cube <方向命令>
方向命令:![alt f,b,u,d,l,r,f_,b_,u_,d_,l_,r_]前后上下左右的英文首字母，不区分大小写，无下划线代表顺时针旋转，有下划线代表逆时针旋转。
